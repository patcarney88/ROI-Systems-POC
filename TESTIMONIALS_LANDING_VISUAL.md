## Color Palette

```
Background:      #0a0a0a (Deep black)
Card Base:       rgba(255,255,255,0.03) (Subtle white overlay)
Card Border:     rgba(255,255,255,0.1) (White 10%)
Card Hover:      rgba(255,255,255,0.05) (White 5%)
Border Hover:    rgba(102,126,234,0.3) (Purple accent)

Text Primary:    rgba(255,255,255,0.9) (Near white)
Text Secondary:  rgba(255,255,255,0.6) (Muted white)

Badge:           rgba(102,126,234,0.1) background
                 rgba(102,126,234,0.3) border
                 #667eea text

Gradient:        linear-gradient(135deg, #667eea 0%, #764ba2 100%)
                 (Purple gradient for accents)

Stars:           #FFB800 (Gold)
```

## Animation & Interactions

1. **Card Hover**
   - Lifts up 8px
   - Border changes from white to purple
   - Purple gradient line appears at top
   - Glowing purple shadow

2. **Star Ratings**
   - Gold color (#FFB800)
   - Drop shadow for visibility
   - Scale up slightly on card hover

3. **Loading Animation**
   - Staggered fade-in (0.1s, 0.2s, 0.3s, 0.4s delays)
   - Respects prefers-reduced-motion

4. **Focus States**
   - Purple outline for keyboard navigation
   - 2px solid with 2px offset
